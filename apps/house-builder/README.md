# 🏠 創意蓋房子遊戲

一個專為 12-15 歲青少年設計的創意建造遊戲，讓玩家自由拖放元件來建造獨一無二的房子。

## ✨ 特色

- 🎨 **自由建造模式** - 無拘無束，發揮創意
- 🎯 **挑戰模式** - 10 個漸進式關卡挑戰
- 📐 **可選網格吸附** - 精確對齊或自由放置
- 🧩 **豐富元件庫** - 41 個精美設計的元件
  - 地基：4 種（草地、沙漠、石板、泥土）
  - 牆壁：8 種（磚牆、木牆、石牆等）
  - 屋頂：6 種（三角頂、平頂、圓頂等）
  - 門：5 種（木門、玻璃門、拱門等）
  - 窗戶：6 種（方窗、圓窗、拱窗等）
  - 裝飾：12 種（樹木、花朵、柵欄、煙囪、路燈等）
- 🎭 **視覺效果** - 流暢的動畫和慶祝特效
- 📱 **響應式設計** - 支援桌面、平板和手機

## 🎮 如何遊玩

### 開始遊戲

1. 打開 `index.html` 在瀏覽器中
2. 選擇遊戲模式（自由建造 或 挑戰模式）
3. 點擊「開始建造」

### 自由建造模式

1. 從左側面板選擇元件
2. 拖放到中央畫布
3. 可以重新拖動已放置的元件調整位置
4. 雙擊元件可以刪除
5. 使用工具欄功能：
   - 📐 切換網格吸附
   - 🗑️ 清空畫布

### 挑戰模式

1. 查看挑戰要求（右側面板）
2. 放置符合要求的元件
3. 點擊「檢查完成」驗證
4. 完成後自動進入下一關卡

## ⌨️ 鍵盤快捷鍵

- `G` - 切換網格吸附
- `Delete` / `Backspace` - 刪除選中的元件
- `Escape` - 取消選擇元件

## 🏗️ 挑戰關卡

1. **基礎房屋** - 1 牆 + 1 屋頂
2. **完整房屋** - 添加門和窗戶
3. **雙牆房屋** - 建造更大的房子
4. **多窗房屋** - 至少 3 扇窗戶
5. **花園別墅** - 添加 3 種裝飾
6. **雙層建築** - 2 層結構
7. **豪華住宅** - 多門多窗
8. **社區設施** - 至少 5 種裝飾
9. **完美莊園** - 大型複合建築
10. **終極挑戰** - 夢想城堡（20+ 元件）

## 🛠️ 技術架構

- **純 Vanilla JavaScript** - 無需框架
- **HTML5 + CSS3** - 現代 Web 標準
- **SVG** - 可縮放的向量圖形
- **Canvas** - 特效層（慶祝動畫）

### 文件結構

```
house-builder/
├── index.html                 # 主 HTML 文件
├── css/
│   ├── main.css              # 主樣式
│   └── animations.css        # 動畫定義
└── js/
    ├── app.js                # 應用入口
    ├── component-library.js  # 元件庫（41 個元件）
    ├── build-engine.js       # 建造引擎
    ├── drag-controller.js    # 拖放控制器
    ├── ui-controller.js      # UI 控制器
    └── challenge-manager.js  # 挑戰管理器
```

## 📝 開發者註記

此遊戲採用模組化架構設計：

- `ComponentLibrary` - 管理所有可用元件
- `BuildEngine` - 處理元件放置、移動、刪除邏輯
- `DragController` - 統一處理滑鼠和觸控拖放事件
- `ChallengeManager` - 管理挑戰關卡和驗證
- `UIController` - 協調所有 UI 更新
